# A Sample Demo with Next.js

Install it and run:

```
npm install
dev:
npm run dev
build:
npm run build
export static page:
npm run export
```

If you want to test the app, install the following serve NPM module globally:
```
npm install serve -g
```

After you've installed "serve", run following commands from the app root:
```
cd out
serve -p 8888<port number>
```
